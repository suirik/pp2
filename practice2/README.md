#  PRACTICE 2: Control Flow & Logic

---

##  Learning Objectives
* Understanding **Boolean** logic and comparison operators.
* Implementing conditional branching with **if-elif-else**.
* Mastering repetition using **for** and **while** loops.
* Controlling loop execution with **break** and **continue**.

---

##  Directory Structure
Detailed breakdown of the files in this practice folder:

### **1. Boolean Logic (`/boolean`)**
* `boolean_intro.py` — Introduction to True/False values.
* `boolean_operators.py` — Using `and`, `or`, and `not`.
* `boolean_comparison.py` — Comparing values (`==`, `!=`, `<`, `>`).

### **2. Conditionals (`/if_else`)**
* `if_statement.py` — Basic if logic.
* `if_else.py` — Handling two paths of execution.
* `if_elif_else.py` — Multiple condition checks.
* `short_hand_if.py` — Writing concise one-line if statements.

### **3. Loops (`/loops`)**
* `while_loop.py` — Condition-based repetition.
* `for_loop.py` — Iterating over sequences and ranges.
* `for_break.py` & `while_break.py` — Stopping loops early.
* `for_continue.py` & `while_continue.py` — Skipping specific iterations.
